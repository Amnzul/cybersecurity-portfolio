About My Home Lab

My home lab is a dedicated environment where I experiment with cybersecurity tools, network configurations, virtualization, and system administration. It serves as a playground for learning, testing, and simulating real-world scenarios safely.

Lab Setup & Hardware

  • Server Hardware: Custom-built server with Intel/AMD CPU, 32GB RAM, SSD and HDD storage.
  • Networking: Managed switch, enterprise firewall, and VLAN segmentation.
  • Virtualization: Proxmox VE for running multiple VMs (Linux, Windows, firewall appliances, etc.).
  • Storage: TrueNAS for network-attached storage and backups.
  • Raspberry Pi Devices: Used for Pi-hole DNS sinkhole and IoT experiments.

Key Projects in the Lab

  • Network Capacity Testing: Simulating and measuring throughput, latency, and bandwidth.
  • Block Ad Trackers: Deploying Pi-hole and firewall rules to block ads and telemetry.
  • TrueNAS Install: Building and configuring a TrueNAS storage server for backups and file sharing.
  • Enterprise Firewall: Deploying pfSense/OPNsense for perimeter defense and traffic monitoring.
  • Port Scan Detection: Setting up Suricata and custom scripts for detecting reconnaissance activity.
  • Packet Analysis: Using tcpdump and Wireshark to analyze network traffic and understand ISP visibility.

What I've Learned

  • Hands-on experience with network architecture, VLANs, and segmentation.
  • Practical skills in deploying and managing enterprise-grade firewalls and IDS/IPS.
  • Automation and scripting with Bash and Python for monitoring and management.
  • Troubleshooting storage solutions and backup strategies.
  • Improved understanding of packet analysis and threat detection.
