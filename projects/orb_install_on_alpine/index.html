Orb Network Monitoring Project Overview

This project involved installing an Orb Sensor on an Alpine Container in Proxmox to monitor network and internet performance.

Software

Orb is a free network monitoring utility for rating internet responsiveness, reliability, and speed available at https://orb.net/

Orb works with a dashboard and a sensor. I will run the app on my mobile phone, and the sensor on an Alpine Linux container in a Proxmox server. Many options are available for where it can be run.

Environment Setup

I used a helper script to quickly install an Alpine Linux container in Proxmox. The script is available at this repository for Proxmox VE Helper Scripts on github.

Run the script provided in the Proxmox shell.

Run the commands in the terminal.

Performance Findings & Analysis

The ratings are all in the green. However, I have a fiber optic connection. I was expecting to see better upload/download numbers. I ran a speedtest through my local ISP's webpage and obtained different results. I suspect the difference is because the test from Orb is relying on a server on a different network for their results.

Conclusions

It could be helpful to have instantaneous insight into network performance if I am experiencing latency somethere.

This can serve as a method to document baseline performance metrics for future comparative analysis.

I will keep the Orb sensor running and get some more data to see what it looks like over time.
