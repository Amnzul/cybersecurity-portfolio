Regular Expression (regex):

# *** Regular expression (regex) *** 

\w matches any alphanumeric character
+ matches any number of occurrences of the character before it.
. matches all characters including symbols.
\. matches to the period character.
\d matches to single digits
\s matches to all single spaces.
 
“b\wa+b” would match “bkaaab”
“\w+xyz” would match strings ending in xyz.
“\d+” would match any numbers.
“\d+{2}” would match any two digit numbers. However if there is a 4 digit number it will see it as two, two digit numbers. A three digit number would capture just the first two digits.
“\d{1,3}” will match a range. The first parameter is the minimum number of repetitions, the second is the max. So {1,3} will match between 1 and 3 digits long.
 
 
“\w+@\w+\.\w+” would match any email addresses.
 
# extract email addresses
import re
email_log =“””paste email here”””
print(re.findall(“\w+@\w+\.\w+”, email_log))
 
 
# findall() requires two parameters, a string containing the regex pattern, and a string to search through.
